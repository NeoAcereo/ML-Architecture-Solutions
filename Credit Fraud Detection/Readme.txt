Different Methods of Dealing with Imbalanced Data

## Data
The credit card fraud detection dataset at https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud was used.

## Methods
1. Resampling
2. SMOTE sampling
3. Metric selection

## Evaluation
Evaluated the learning performance of the sampling methods(Resampling and SMOTE) on a test set using the following metrics:
1. Accuracy
2. Precision
3. Recall
